# Contact Management System in C

## Overview

The Contact Management System is a console-based application developed in the C programming language. It allows users to efficiently manage contact information through a simple menu-driven interface. The system provides essential functionalities such as adding new contacts, viewing all saved contacts, searching for specific contacts, and deleting existing contacts.

This project demonstrates the practical use of C programming concepts including structures, arrays, functions, loops, conditional statements, and string handling. It is designed as an intermediate-level project suitable for students learning data management and file organization concepts in C.

## Features

* Add new contacts with name, phone number, and email address
* View all stored contacts
* Search contacts by name
* Delete contacts from the system
* User-friendly menu-driven interface
* Efficient contact storage using structures

## Technologies Used

* Programming Language: C
* Concepts: Structures, Arrays, Functions, String Manipulation, Menu-Driven Programming

## Learning Outcomes

This project helps in understanding:

* Data organization using structures
* Modular programming with functions
* Searching and deleting records
* Managing collections of data in C
* Building real-world console applications

## Future Enhancements

* File handling for permanent data storage
* Contact editing functionality
* Sorting contacts alphabetically
* Password protection
* Dynamic memory allocation
* Advanced search and filtering options

This project serves as a strong foundation for developing more advanced database and management systems in C.
